# Personalized Health Recommendation System

A machine learning-powered healthcare web application that provides personalized lifestyle recommendations based on user health information.

## Features
- Personalized diet recommendations
- Exercise guidance
- Water intake estimation
- Sleep optimization suggestions


## Live Demo 

🔗 [Health Recommendation System](https://health-recommendation-system-kkj8.onrender.com)


## Technologies Used
- Python
- Flask
- HTML
- CSS
- Machine Learning
